---
title: "Conversation"
description: "API reference for the Conversation enum in XMTP iOS SDK"
---

The `Conversation` enum provides a unified interface for working with both group conversations and DMs. It wraps either a `Group` or `Dm` and provides common methods that work with both types.

## Overview

```swift
public enum Conversation {
    case group(Group)
    case dm(Dm)
}
```

Use `Conversation` when you want to handle both groups and DMs uniformly without needing to check the specific type.

## Type Checking

### type

```swift
public var type: XMTPConversationType
```

Get the conversation type.

```swift
public enum XMTPConversationType {
    case group
    case dm
}
```

```swift
switch conversation.type {
case .group:
    print("This is a group")
case .dm:
    print("This is a DM")
}
```

### Extracting Specific Types

You can pattern match to access the underlying type:

```swift
switch conversation {
case .group(let group):
    // Access group-specific properties
    let name = try group.name()
case .dm(let dm):
    // Access DM-specific properties
    let peerId = try dm.peerInboxId
}
```

## Retrieving Messages

### messages()

Get messages from the conversation.

```swift
public func messages(
    limit: Int? = nil,
    beforeNs: Int64? = nil,
    afterNs: Int64? = nil,
    direction: SortDirection? = .descending,
    deliveryStatus: MessageDeliveryStatus = .all,
    excludeContentTypes: [StandardContentType]? = nil,
    excludeSenderInboxIds: [String]? = nil,
    sortBy: MessageSortBy? = nil,
    insertedAfterNs: Int64? = nil,
    insertedBeforeNs: Int64? = nil
) async throws -> [DecodedMessage]
```

<ParamField path="limit" type="Int" optional>
  Maximum number of messages to return
</ParamField>

<ParamField path="beforeNs" type="Int64" optional>
  Messages sent before this timestamp
</ParamField>

<ParamField path="afterNs" type="Int64" optional>
  Messages sent after this timestamp
</ParamField>

<ParamField path="direction" type="SortDirection" default=".descending">
  Sort direction
</ParamField>

<ParamField path="deliveryStatus" type="MessageDeliveryStatus" default=".all">
  Filter by delivery status
</ParamField>

<ParamField path="excludeContentTypes" type="[StandardContentType]" optional>
  Content types to exclude
</ParamField>

<ParamField path="excludeSenderInboxIds" type="[String]" optional>
  Sender inbox IDs to exclude
</ParamField>

<ParamField path="sortBy" type="MessageSortBy" optional>
  Sort by field
</ParamField>

<ParamField path="insertedAfterNs" type="Int64" optional>
  Filter by insertion time (after)
</ParamField>

<ParamField path="insertedBeforeNs" type="Int64" optional>
  Filter by insertion time (before)
</ParamField>

<ResponseField name="messages" type="[DecodedMessage]">
  Array of decoded messages
</ResponseField>

```swift
let messages = try await conversation.messages(limit: 50)
```

<Info>
  For UI rendering, consider using `enrichedMessages()` instead, which includes reactions, replies, and other metadata.
</Info>

### enrichedMessages()

Get messages with enriched metadata.

```swift
public func enrichedMessages(
    limit: Int? = nil,
    beforeNs: Int64? = nil,
    afterNs: Int64? = nil,
    direction: SortDirection? = .descending,
    deliveryStatus: MessageDeliveryStatus = .all,
    excludeContentTypes: [StandardContentType]? = nil,
    excludeSenderInboxIds: [String]? = nil,
    sortBy: MessageSortBy? = nil,
    insertedAfterNs: Int64? = nil,
    insertedBeforeNs: Int64? = nil
) async throws -> [DecodedMessageV2]
```

Parameters are the same as `messages()`.

<ResponseField name="messages" type="[DecodedMessageV2]">
  Array of enriched messages with reactions, replies, etc.
</ResponseField>

```swift
let messages = try await conversation.enrichedMessages(limit: 50)
for message in messages {
    print("Message: \(message.content())")
    print("Reactions: \(message.reactions.count)")
}
```

<Note>
  Recommended for UI rendering. This automatically includes reactions, replies, and other associated data.
</Note>

## Example Usage

```swift
// Get a conversation (could be group or DM)
let conversation = try await client.conversations.list().first!

// Send a message (works for both groups and DMs)
try await conversation.send(content: "Hello!")

// Get messages
let messages = try await conversation.enrichedMessages(limit: 50)

// Stream new messages
for try await message in conversation.streamMessages() {
    print("New: \(message.content())")
}

// Check the type if needed
switch conversation.type {
case .group:
    print("This is a group conversation")
case .dm:
    print("This is a direct message")
}
```
