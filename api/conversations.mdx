title: "Conversations"
description: "API reference for managing conversations in XMTP iOS SDK"

The `Conversations` class provides methods for creating, listing, and managing conversations (both groups and DMs).

Access via `client.conversations`.

## Creating Conversations

### newGroup()

Create a new group conversation with inbox IDs.

```swift
public func newGroup(
    with inboxIds: [InboxId],
    permissions: GroupPermissionPreconfiguration = .allMembers,
    name: String = "",
    imageUrl: String = "",
    description: String = "",
    disappearingMessageSettings: DisappearingMessageSettings? = nil,
    appData: String? = nil
) async throws -> Group
```

<ParamField path="inboxIds" type="[String]" required>
  Array of inbox IDs to add to the group
</ParamField>

<ParamField path="permissions" type="GroupPermissionPreconfiguration" default=".allMembers">
  Permission preset for the group (.allMembers or .adminOnly)
</ParamField>

<ParamField path="name" type="String" default='""'>
  Group name
</ParamField>

<ParamField path="imageUrl" type="String" default='""'>
  Group image URL (square)
</ParamField>

<ParamField path="description" type="String" default='""'>
  Group description
</ParamField>

<ParamField path="disappearingMessageSettings" type="DisappearingMessageSettings" optional>
  Disappearing message settings for the group
</ParamField>

<ParamField path="appData" type="String" optional>
  Custom application data
</ParamField>

<ResponseField name="group" type="Group">
  The newly created group
</ResponseField>

```swift
let group = try await client.conversations.newGroup(
    with: ["inbox1", "inbox2"],
    permissions: .allMembers,
    name: "Dev Team",
    imageUrl: "https://example.com/image.png",
    description: "Discussion for developers"
)
```

### newGroupWithIdentities()

Create a new group conversation with public identities.

```swift
public func newGroupWithIdentities(
    with identities: [PublicIdentity],
    permissions: GroupPermissionPreconfiguration = .allMembers,
    name: String = "",
    imageUrl: String = "",
    description: String = "",
    disappearingMessageSettings: DisappearingMessageSettings? = nil,
    appData: String? = nil
) async throws -> Group
```

<ParamField path="identities" type="[PublicIdentity]" required>
  Array of public identities to add to the group
</ParamField>

Other parameters are the same as `newGroup()`.

```swift

let identity2 = PublicIdentity(kind: .ethereum, identifier: "0x456...")

let group = try await client.conversations.newGroupWithIdentities(
    with: [identity1, identity2],
    name: "Friends"
)
```

</ParamField>

</ParamField>

## Listing Conversations

### list()

List all conversations (both groups and DMs).

```swift
public func list(
    createdAfterNs: Int64? = nil,
    createdBeforeNs: Int64? = nil,
    lastActivityAfterNs: Int64? = nil,
    lastActivityBeforeNs: Int64? = nil,
    limit: Int? = nil,
    consentStates: [ConsentState]? = nil,
    orderBy: ConversationsOrderBy = .lastActivity
) async throws -> [Conversation]
```

<ParamField path="createdAfterNs" type="Int64" optional>
  Filter to conversations created after this timestamp (nanoseconds)
</ParamField>

<ParamField path="createdBeforeNs" type="Int64" optional>
  Filter to conversations created before this timestamp
</ParamField>

<ParamField path="lastActivityAfterNs" type="Int64" optional>
  Filter to conversations with activity after this timestamp
</ParamField>

<ParamField path="lastActivityBeforeNs" type="Int64" optional>
  Filter to conversations with activity before this timestamp
</ParamField>

<ParamField path="limit" type="Int" optional>
  Maximum number of conversations to return
</ParamField>

<ParamField path="consentStates" type="[ConsentState]" optional>
  Filter by consent state (.allowed, .denied, .unknown)
</ParamField>

<ParamField path="orderBy" type="ConversationsOrderBy" default=".lastActivity">
  Sort order (.createdAt or .lastActivity)
</ParamField>

<ResponseField name="conversations" type="[Conversation]">
  Array of conversations
</ResponseField>

```swift
let conversations = try await client.conversations.list(
    limit: 20,
    consentStates: [.allowed],
    orderBy: .lastActivity
)
```

## Streaming

### stream()

Stream new conversations as they are created.

```swift
public func stream(
    type: ConversationFilterType = .all,
    onClose: (() -> Void)? = nil
) -> AsyncThrowingStream<Conversation, Error>
```

<ParamField path="type" type="ConversationFilterType" default=".all">
  Filter type (.all, .groups, or .dms)
</ParamField>

<ParamField path="onClose" type="(() -> Void)" optional>
  Callback when the stream closes
</ParamField>

<ResponseField name="stream" type="AsyncThrowingStream<Conversation, Error>">
  Stream of new conversations
</ResponseField>

```swift
for try await conversation in client.conversations.stream() {
    print("New conversation: \(conversation.id)")
}
```

### streamAllMessages()

Stream all messages from all conversations.

```swift
public func streamAllMessages(
    type: ConversationFilterType = .all,
    consentStates: [ConsentState]? = nil,
    onClose: (() -> Void)? = nil
) -> AsyncThrowingStream<DecodedMessage, Error>
```

<ParamField path="type" type="ConversationFilterType" default=".all">
  Filter type (.all, .groups, or .dms)
</ParamField>

<ParamField path="consentStates" type="[ConsentState]" optional>
  Filter by consent states
</ParamField>

<ParamField path="onClose" type="(() -> Void)" optional>
  Callback when the stream closes
</ParamField>

<ResponseField name="stream" type="AsyncThrowingStream<DecodedMessage, Error>">
  Stream of all new messages
</ResponseField>

```swift
for try await message in client.conversations.streamAllMessages() {
    print("New message from \(message.senderInboxId): \(message.content())")
}
```

## Types

### ConversationFilterType

```swift
public enum ConversationFilterType {
    case all
    case groups
    case dms
}
```

Filter type for conversation operations.

### ConversationsOrderBy

```swift
public enum ConversationsOrderBy {
    case createdAt
    case lastActivity
}
```

Sort order for listing conversations.

### GroupSyncSummary

```swift
public struct GroupSyncSummary {
    public var numEligible: UInt64
    public var numSynced: UInt64
}
```

Summary of a sync operation.

### ConversationError

```swift
public enum ConversationError: Error {
    case memberCannotBeSelf
    case memberNotRegistered([String])
    case groupsRequireMessagePassed
    case notSupportedByGroups
    case streamingFailure
}
```

Errors that can be thrown by conversation operations.
