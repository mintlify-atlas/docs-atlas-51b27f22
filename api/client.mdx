The `Client` class is the main entry point for interacting with the XMTP network. It manages authentication, conversations, and network connections.

## Creating a Client

### create()

Create a new XMTP client with a signing key.

```swift
public static func create(
    account: SigningKey,
    options: ClientOptions
) async throws -> Client
```

<ParamField path="account" type="SigningKey" required>
  The signing key used to authenticate and sign messages
</ParamField>

<ParamField path="options" type="ClientOptions" required>
  Configuration options for the client
</ParamField>

<ResponseField name="Client" type="Client">
  Returns an initialized and authenticated XMTP client
</ResponseField>

```swift
let client = try await Client.create(
    account: signingKey,
    options: clientOptions
)
```

### build()

Build a client without signing capabilities (read-only mode).

```swift
public static func build(
    publicIdentity: PublicIdentity,
    options: ClientOptions,
    inboxId: InboxId? = nil
) async throws -> Client
```

<ParamField path="publicIdentity" type="PublicIdentity" required>
  The public identity to build the client for
</ParamField>

<ParamField path="options" type="ClientOptions" required>
  Configuration options for the client
</ParamField>

<ParamField path="inboxId" type="String" optional>
  Optional inbox ID. If provided, builds offline without network calls
</ParamField>

<ResponseField name="Client" type="Client">
  Returns a client instance without signing capabilities
</ResponseField>

```swift
let client = try await Client.build(
    publicIdentity: publicIdentity,
    options: clientOptions
)
```

## Static Methods

### canMessage()

Check if accounts can receive XMTP messages.

```swift
public static func canMessage(
    accountIdentities: [PublicIdentity],
    api: ClientOptions.Api
) async throws -> [String: Bool]
```

<ParamField path="accountIdentities" type="[PublicIdentity]" required>
  Array of public identities to check
</ParamField>

<ParamField path="api" type="ClientOptions.Api" required>
  API configuration to use for the check
</ParamField>

<ResponseField name="result" type="[String: Bool]">
  Dictionary mapping identity identifiers to boolean indicating if they can receive messages
</ResponseField>

```swift
let canMessageMap = try await Client.canMessage(
    accountIdentities: [identity1, identity2],
    api: clientOptions.api
)
print(canMessageMap["0x123..."]) // true or false
```

### addAccount()

Add a new account to this inbox.

```swift
public func addAccount(
    newAccount: SigningKey,
    allowReassignInboxId: Bool = false

```

<ParamField path="newAccount" type="SigningKey" required>

</ParamField>

<ParamField path="allowReassignInboxId" type="Bool" default="false">

</ParamField>

<Warning>

## ClientError

Errors that can be thrown by Client operations.

```swift
public enum ClientError: Error {
    case creationError(String)
    case missingInboxId
    case invalidInboxId(String)
}
```

### creationError

Thrown when client creation fails.

### missingInboxId

Thrown when an inbox ID is required but not available.

### invalidInboxId

Thrown when an inbox ID format is invalid (e.g., starts with '0x').

## ClientOptions

Configuration options for creating a client.

```swift
public struct ClientOptions {
    public var api: Api = Api()
    public var codecs: [any ContentCodec] = []
    public var preAuthenticateToInboxCallback: PreEventCallback?
    public var dbEncryptionKey: Data
    public var dbDirectory: String?
    public var historySyncUrl: String?
    public var deviceSyncEnabled: Bool
    public var debugEventsEnabled: Bool
    public var forkRecoveryOptions: ForkRecoveryOptions?
}
```

<ParamField path="api" type="Api" default="Api()">
  API configuration (environment, security settings)
</ParamField>

<ParamField path="codecs" type="[ContentCodec]" default="[]">
  Array of content codecs to register
</ParamField>

<ParamField path="preAuthenticateToInboxCallback" type="PreEventCallback" optional>
  Callback invoked before authentication
</ParamField>

<ParamField path="dbEncryptionKey" type="Data" required>
  Encryption key for the local database
</ParamField>

<ParamField path="dbDirectory" type="String" optional>
  Custom directory for the database (defaults to documents directory)
</ParamField>

<ParamField path="historySyncUrl" type="String" optional>
  URL for history sync server
</ParamField>

<ParamField path="deviceSyncEnabled" type="Bool" default="true">
  Whether device sync is enabled
</ParamField>

<ParamField path="debugEventsEnabled" type="Bool" default="false">
  Whether debug events are enabled
</ParamField>

<ParamField path="forkRecoveryOptions" type="ForkRecoveryOptions" optional>
  Fork recovery configuration
</ParamField>

### ClientOptions.Api

Network configuration options.

```swift
public struct Api {
    public var env: XMTPEnvironment = .dev
    public var isSecure: Bool = true
    public var appVersion: String?
    public var gatewayHost: String?
}
```

<ParamField path="env" type="XMTPEnvironment" default=".dev">
  Network environment (.dev, .production, .local)
</ParamField>

<ParamField path="isSecure" type="Bool" default="true">
  Whether to use TLS (should only be false for .local)
</ParamField>

<ParamField path="appVersion" type="String" optional>
  App version string
</ParamField>

<ParamField path="gatewayHost" type="String" optional>
  Custom gateway host URL
</ParamField>

```swift
let options = ClientOptions(
    api: ClientOptions.Api(
        env: .production,
        isSecure: true
    ),
    dbEncryptionKey: encryptionKey,
    deviceSyncEnabled: true
)
```
