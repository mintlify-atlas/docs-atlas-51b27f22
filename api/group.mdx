---
title: "Group"
description: "API reference for Group conversations in XMTP iOS SDK"
---

The `Group` struct represents a group conversation in XMTP. Groups support multiple members, permissions, and rich metadata.

## Properties

### id

```swift
public var id: String
```

The unique hex-encoded identifier for the group.

### topic

```swift
public var topic: String
```

The topic string for push notifications and streaming.

### createdAt

```swift
public var createdAt: Date
```

The date when the group was created.

### createdAtNs

```swift
public var createdAtNs: Int64
```

The creation timestamp in nanoseconds.

### lastActivityAtNs

```swift
public var lastActivityAtNs: Int64
```

The timestamp of the last activity (message sent) in nanoseconds.

### members

```swift
public var members: [Member] { get async throws }
```

The list of group members.

```swift
let members = try await group.members
for member in members {
    print("Member: \(member.inboxId)")
}
```

### peerInboxIds

```swift
public var peerInboxIds: [InboxId] { get async throws }
```

The list of peer inbox IDs (excluding the current user).

### membershipState

```swift

```

## Sending Messages

### send()

Send a message to the group.

```swift
public func send<T>(
    content: T,
    options: SendOptions? = nil
) async throws -> String
```

<ParamField path="content" type="T" required>
  The content to send (type depends on registered codecs)
</ParamField>

<ParamField path="options" type="SendOptions" optional>
  Send options (compression, content type, ephemeral)
</ParamField>

<ResponseField name="messageId" type="String">
  The hex-encoded message ID
</ResponseField>

```swift
// Send text
let messageId = try await group.send(content: "Hello group!")

// Send with options
let messageId = try await group.send(
    content: attachment,
    options: SendOptions(contentType: AttachmentCodec.contentType)
)
```

### prepareMessage()

Prepare a message for sending (optimistic send).

```swift
public func prepareMessage<T>(
    content: T,
    options: SendOptions? = nil,
    noSend: Bool = false
) async throws -> String
```

<ParamField path="content" type="T" required>
  The content to prepare
</ParamField>

<ParamField path="options" type="SendOptions" optional>
  Send options
</ParamField>

<ParamField path="noSend" type="Bool" default="false">
  If true, only prepare without sending
</ParamField>

<ResponseField name="messageId" type="String">
  The hex-encoded message ID
</ResponseField>

```swift
// Prepare and send optimistically
let messageId = try await group.prepareMessage(content: "Hello")

// Just prepare without sending
let messageId = try await group.prepareMessage(
    content: "Hello",
    noSend: true
)
```

## Retrieving Messages

### messages()

Get messages from the group.

```swift
public func messages(
    beforeNs: Int64? = nil,
    afterNs: Int64? = nil,
    limit: Int? = nil,
    direction: SortDirection? = .descending,
    deliveryStatus: MessageDeliveryStatus = .all,
    excludeContentTypes: [StandardContentType]? = nil,
    excludeSenderInboxIds: [String]? = nil,
    sortBy: MessageSortBy? = nil,
    insertedAfterNs: Int64? = nil,
    insertedBeforeNs: Int64? = nil
) async throws -> [DecodedMessage]
```

<ParamField path="beforeNs" type="Int64" optional>
  Get messages sent before this timestamp
</ParamField>

<ParamField path="afterNs" type="Int64" optional>
  Get messages sent after this timestamp
</ParamField>

<ParamField path="limit" type="Int" optional>
  Maximum number of messages to return
</ParamField>

<ParamField path="direction" type="SortDirection" default=".descending">
  Sort direction (.ascending or .descending)
</ParamField>

<ParamField path="deliveryStatus" type="MessageDeliveryStatus" default=".all">
  Filter by delivery status (.published, .unpublished, .failed, .all)
</ParamField>

<ParamField path="excludeContentTypes" type="[StandardContentType]" optional>
  Content types to exclude
</ParamField>

<ParamField path="excludeSenderInboxIds" type="[String]" optional>
  Sender inbox IDs to exclude
</ParamField>

<ParamField path="sortBy" type="MessageSortBy" optional>
  Sort by field
</ParamField>

<ParamField path="insertedAfterNs" type="Int64" optional>
  Filter by database insertion time (after)
</ParamField>

<ParamField path="insertedBeforeNs" type="Int64" optional>
  Filter by database insertion time (before)
</ParamField>

<ResponseField name="messages" type="[DecodedMessage]">
  Array of decoded messages
</ResponseField>

```swift
let messages = try await group.messages(limit: 50)
for message in messages {
    print(message.content())
}
```

<Info>
  This method returns raw messages without enriched metadata. For UI rendering, consider using `enrichedMessages()` instead.
</Info>

### enrichedMessages()

Get messages with enriched metadata (reactions, replies, etc.).

```swift
public func enrichedMessages(
    beforeNs: Int64? = nil,
    afterNs: Int64? = nil,
    limit: Int? = nil,
    direction: SortDirection? = .descending,
    deliveryStatus: MessageDeliveryStatus = .all,
    excludeContentTypes: [StandardContentType]? = nil,
    excludeSenderInboxIds: [String]? = nil,
    sortBy: MessageSortBy? = nil,
    insertedAfterNs: Int64? = nil,
    insertedBeforeNs: Int64? = nil
) async throws -> [DecodedMessageV2]
```

Parameters are the same as `messages()`.

<ResponseField name="messages" type="[DecodedMessageV2]">
  Array of enriched decoded messages
</ResponseField>

```swift
let messages = try await group.enrichedMessages(limit: 50)
for message in messages {
    print("Message: \(message.content())")
    print("Reactions: \(message.reactions.count)")
    if let reply = message.replyToMessage {
        print("Reply to: \(reply.content())")
    }
}
```

<Note>
  Recommended for UI rendering. This method includes reactions, replies, and other metadata automatically.
</Note>
