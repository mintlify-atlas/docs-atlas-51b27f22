Direct messages (DMs) provide private, one-on-one conversations between two participants. DMs are simpler than group chats but still support rich features like disappearing messages and content types.

## Creating DMs

### Create or Find a DM

The recommended method creates a new DM or returns an existing one:

```swift DmTests.swift:50-60
let dm = try await client.conversations.findOrCreateDm(
    with: recipientInboxId
)

print("DM ID: \(dm.id)")
```

This is idempotent - calling it multiple times with the same inbox ID returns the same DM.

### Create with Identity

Create a DM using a wallet address:

```swift DmTests.swift:63-75
let dm = try await client.conversations.findOrCreateDmWithIdentity(
    with: PublicIdentity(
        kind: .ethereum,
        identifier: "0x123..."
    )
)
```

### Create New Conversation

Force creation of a new conversation:

```swift
let dm = try await client.conversations.newConversation(
    with: inboxId
)
```

Or with identity:

```swift
let dm = try await client.conversations.newConversationWithIdentity(
    with: identity
)
```

## Validation and Errors

### Cannot DM Yourself

```swift DmTests.swift:91-100
// This will throw an error
do {
    let dm = try await client.conversations.findOrCreateDm(
        with: client.inboxID
    )
} catch {
    print("Cannot create DM with yourself")
}
```

### Recipient Must Be Registered

```swift DmTests.swift:123-133
// Check if identity can receive messages first
let canMessage = try await client.canMessage(
    identity: recipientIdentity
)

if canMessage {
    let dm = try await client.conversations.findOrCreateDmWithIdentity(
        with: recipientIdentity
    )
} else {
    print("Recipient is not registered on XMTP")
}
```

### Use Inbox IDs, Not Addresses

```swift DmTests.swift:102-121
// WRONG - Don't use wallet addresses directly
do {
    let dm = try await client.conversations.newConversation(
        with: "0x123..."  // Address, not inbox ID
    )
} catch ClientError.invalidInboxId(let id) {
    print("Invalid inbox ID: \(id)")
}

// CORRECT - Use inbox ID
let inboxId = try await client.inboxIdFromIdentity(
    identity: recipientIdentity
)

let dm = try await client.conversations.newConversation(
    with: inboxId!
)
```

## Best Practices

<AccordionGroup>
  <Accordion title="Use findOrCreateDm">
    Always use `findOrCreateDm` to avoid duplicate conversations:
    
    ```swift
    // RECOMMENDED
    let dm = try await client.conversations.findOrCreateDm(
        with: recipientInboxId
    )
    
    // NOT RECOMMENDED - may create duplicates
    let dm = try await client.conversations.newConversation(
        with: recipientInboxId
    )
    ```
  </Accordion>

  <Accordion title="Check registration status">
    Verify recipients can receive messages:
    
    ```swift
    let canMessage = try await client.canMessage(
        identity: recipientIdentity
    )
    
    if canMessage {
        let dm = try await client.conversations.findOrCreateDmWithIdentity(
            with: recipientIdentity
        )
    } else {
        showAlert("Recipient is not on XMTP")
    }
    ```
  </Accordion>

  <Accordion title="Sync before reading">
    Sync the DM before accessing messages:
    
    ```swift
    try await dm.sync()
    let messages = try await dm.messages()
    ```
  </Accordion>

  <Accordion title="Use enrichedMessages for UI">
    For chat interfaces, use enriched messages:
    
    ```swift
    let messages = try await dm.enrichedMessages(
        limit: 50,
        direction: .descending
    )
    
    // Reactions and replies are included
    for message in messages {
        displayMessage(message)
    }
    ```
  </Accordion>

  <Accordion title="Handle consent properly">
    Respect user consent preferences:
    
    ```swift
    let dm = try await client.conversations.findOrCreateDm(
        with: recipientInboxId
    )
    
    // Check consent before showing in main list
    let consentState = try dm.consentState()
    
    switch consentState {
    case .allowed:
        showInMainList(dm)
    case .denied:
        showInBlockedList(dm)
    case .unknown:
        showInRequestsList(dm)
    }
    ```
  </Accordion>
</AccordionGroup>

## Examples

### Create DM with Validation

```swift
func createDM(
    with recipientIdentity: PublicIdentity
) async throws -> Dm? {
    // Check if recipient can receive messages
    let canMessage = try await client.canMessage(
        identity: recipientIdentity
    )
    
    guard canMessage else {
        print("Recipient is not registered on XMTP")
        return nil
    }
    
    // Get or create inbox ID
    guard let inboxId = try await client.inboxIdFromIdentity(
        identity: recipientIdentity
    ) else {
        print("Could not resolve inbox ID")
        return nil
    }
    
    // Cannot DM yourself
    guard inboxId != client.inboxID else {
        print("Cannot create DM with yourself")
        return nil
    }
    
    // Create or find DM
    let dm = try await client.conversations.findOrCreateDm(
        with: inboxId
    )
    
    return dm
}
```

### DM Chat View Model

```swift
class DMViewModel: ObservableObject {
    @Published var messages: [DecodedMessageV2] = []
    @Published var peerName: String = ""
    
    let dm: Dm
    let client: Client
    private var streamTask: Task<Void, Never>?
    
    init(dm: Dm, client: Client) {
        self.dm = dm
        self.client = client
    }
    
    func load() async {
        do {
            // Sync first
            try await dm.sync()
            
            // Load messages
            messages = try await dm.enrichedMessages(
                limit: 50,
                direction: .descending
            )
            
            // Get peer name
            let peerInboxId = try dm.peerInboxId
            peerName = await resolveName(for: peerInboxId)
            
            // Start streaming
            startStreaming()
            
        } catch {
            print("Failed to load DM: \(error)")
        }
    }
    
    func startStreaming() {
        streamTask = Task { [weak self] in
            guard let self = self else { return }
            
            let stream = dm.streamMessages()
            
            for try await message in stream {
                guard !Task.isCancelled else { break }
                
                await MainActor.run {
                    let enriched = DecodedMessageV2.create(
                        ffiMessage: message
                    )
                    self.messages.insert(enriched, at: 0)
                }
            }
        }
    }
    
    func sendMessage(_ text: String) async {
        do {
            _ = try await dm.send(content: text)
        } catch {
            print("Failed to send message: \(error)")
        }
    }
    
    func loadMore() async {
        guard let oldest = messages.last else { return }
        
        let more = try? await dm.enrichedMessages(
            beforeNs: oldest.sentAtNs,
            limit: 50
        )
        
        if let more = more {
            await MainActor.run {
                messages.append(contentsOf: more)
            }
        }
    }
    
    func resolveName(for inboxId: String) async -> String {
        // Implement your name resolution logic
        // e.g., ENS lookup, address book, etc.
        return inboxId.prefix(8) + "..."
    }
    
    deinit {
        dm.endStream()
        streamTask?.cancel()
    }
}
```

### Message Request Handler

```swift
class MessageRequestHandler {
    let client: Client
    
    init(client: Client) {
        self.client = client
    }
    
    func getPendingRequests() async throws -> [Dm] {
        // Get DMs with unknown consent state
        return try await client.conversations.listDms(
            consentStates: [.unknown]
        )
    }
    
    func acceptRequest(dm: Dm) async throws {
        try await dm.updateConsentState(state: .allowed)
        print("Accepted message request")
    }
    
    func declineRequest(dm: Dm) async throws {
        try await dm.updateConsentState(state: .denied)
        print("Declined message request")
    }
    
    func blockDM(dm: Dm) async throws {
        try await dm.updateConsentState(state: .denied)
        print("Blocked DM")
    }
    
    func unblockDM(dm: Dm) async throws {
        try await dm.updateConsentState(state: .allowed)
        print("Unblocked DM")
    }
}
```
