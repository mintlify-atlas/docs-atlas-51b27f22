<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with XMTP in minutes
  </p>

  <Steps>
    <Step title="Install the SDK">
      Add XMTP iOS SDK to your project using Swift Package Manager.

      ```swift Package.swift
      dependencies: [
        .package(url: "https://github.com/xmtp/xmtp-ios", from: "4.0.0")
      ]
      ```
    </Step>

    <Step title="Create a client">
      Initialize an XMTP client with a wallet signer.

      ```swift
      import XMTPiOS

      let client = try await Client.create(
        account: wallet,
        options: .init(
          api: .init(env: .production)
        )
      )
      ```
    </Step>

    <Step title="Start a conversation">
      Create a group chat or direct message to begin messaging.

      ```swift
      let group = try await client.conversations.newGroup(
        with: [peerAddress],
        permissions: .allMembers
      )

      try await group.send(content: "Hello, XMTP!")
      ```
    </Step>

    <Step title="Receive messages">
      Stream incoming messages in real-time.

      ```swift
      for try await message in group.streamMessages() {
        print("New message: \(message.content())")
      }
      ```
    </Step>
  </Steps>
</div>
